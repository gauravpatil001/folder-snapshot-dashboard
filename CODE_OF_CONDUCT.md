# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the community

Examples of unacceptable behavior:
- Harassment or discriminatory language
- Personal attacks
- Publishing private information without permission

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, and discussions.

## Enforcement

To report unacceptable behavior, contact the maintainer via GitHub issues or direct repository contact channels.

Maintainers may remove, edit, or reject comments, commits, code, and other contributions that are not aligned with this Code of Conduct.
